## I. Demo Script

Hi, this is Smart Content Purifier & Summary Hub, a lightweight AI utility MVP built with Python and Gradio.

The goal of this project is to help users prepare messy or long-form text so it becomes easier to read, summarize, and reuse. Many people copy text from AI drafts, meeting notes, study materials, or articles, but the result is often too long or not well structured.

In this app, the user can paste raw text into the input box, then choose one of three modes: Clean Text, Short Summary, or Key Points. Clean Text improves the formatting and readability. Short Summary creates a shorter version of the content. Key Points turns the main ideas into a bullet list.

After generating the output, the user can also click Export Markdown to get a clean Markdown version that is ready to copy into notes, documentation, or content workflows.

The project is deployed on Hugging Face Spaces and uses a lightweight rule-based approach, so it can run without paid APIs, private credentials, or heavy model dependencies. For this MVP, the focus was on stability, public deployment, clean structure, and clear documentation.
