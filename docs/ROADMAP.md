# Roadmap

This roadmap keeps **Smart Content Purifier & Summary Hub** focused, lightweight, and portfolio-ready.

## Current Stage

### MVP v0.1 — Live

Status: **Completed / Live**

Included features:

- Clean Text
- Short Summary
- Key Points
- Export Markdown
- Gradio UI
- Hugging Face Spaces deployment
- Basic GitHub project structure

## Out of Scope for Now

The following items are intentionally excluded from the current MVP path:

- Paid APIs
- Private API keys
- CRM integration
- HubSpot integration
- Lead scoring
- Contact CSV pipeline
- Landing page bonus features
- Mixing with other projects such as `pusaka-ebook-ai-agent`

## Guiding Principle

Keep the project simple, stable, public, and easy to understand. A working portfolio MVP is more valuable than a large but unstable feature set.
